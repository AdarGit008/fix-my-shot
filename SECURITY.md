# Security Policy

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue.

- **Preferred:** open a private advisory via GitHub Security Advisories —
  <https://github.com/AdarGit008/fix-my-shot/security/advisories/new>
- **Email:** adar008@gmail.com

Please include steps to reproduce and the impact you observed. You'll get an
acknowledgement within a few days. Once a fix is available it will be noted in
the [CHANGELOG](CHANGELOG.md).

## Supported versions

This project is in an early **ideation / prototype** phase and has not cut a
release. There is no supported release line yet; the default branch (`main`) is
the only line that receives fixes. This section will be updated when the first
version is tagged.

| Version | Supported |
| ------- | --------- |
| `main` (unreleased) | ✅ |
